How to Use

    

        
Getting Started

        
1. Add your Gemini API Key in the "API Configuration" section and click Save.

        
2. Open the extension popup (by clicking the icon in your toolbar) and click "Start Session".

        
3. As you browse, the extension will automatically log and summarize pages.

        
4. Click "Stop Session" when you're done. You can click "Summarize Session" to get a report.

        
Custom Rules

        
Use a URL pattern with wildcards (*) to match sites. For fields, provide a JSON object where keys are the names of the data you want, and values are a brief description of where to find that data on the page (e.g., `{'article_author': 'the name of the author of the post'}`).
